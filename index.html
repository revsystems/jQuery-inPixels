jQuery-inPixels

This demo shows some simple return values of margins, paddings and borders which are usable for javascript calculations.

It supports numeric widths in all browsers even when the css sets them to "thin", "medium", "thick", or "auto".

Example Box

Get left padding
Get left border

Get margin width of inner_container
Get padding width
Get border width

Get top margin
Get top padding
Get top border

Get margin height
Get padding height
Get border height

There is also some built-in functionality to get any css measurement as a pixel value. For em/ex, this is calculated in the global scope (immediately under the body tag).
